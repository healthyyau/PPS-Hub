## Token Usage Checklist

Before marking a component as Done:

- [ ] All colors use `var(--color-*)` semantic tokens
- [ ] All typography uses `var(--typography-*)`
- [ ] All spacing uses `var(--spacing-*)`
- [ ] No hardcoded hex, px, or font values
- [ ] Dark mode tested
- [ ] TypeScript compiles with `npm run build`
- [ ] JSDoc comment on component
